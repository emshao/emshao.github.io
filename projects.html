Combat Robotics

[Spring/Fall 2022] Competed in a just-for-fun mini battlebots competition in which my two teammates and I built a one-pound RC-controlled robot with holonomic drive. I work on features including reading in and analyzing data from an IMU to integrate field-oriented driving, implementing PID loops to control auto adjustment, and programming basic Kiwi Drive.

Color Detection Robot

[Spring 2022] Built and programming a color and light sensing robot using an RBG color sensor to detech the presence of mirrors and obstacles within the color rings. Also implemented core functionality such as black line following, automatic adjustment of motor control based on light sensor feedback, and logic to display information about what it scanned.
